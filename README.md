# Demo Repository

This repository contains various demo files and utilities.

## Palindrome Checker

A Python utility to check if strings are palindromes.

### Usage

```bash
python3 palindrome.py
```

The script will:
1. Run through example test cases
2. Enter interactive mode where you can test your own strings

### What is a Palindrome?

A palindrome is a word, phrase, number, or sequence that reads the same forward and backward, ignoring spaces, punctuation, and case.

Examples:
- "racecar" ✓
- "A man a plan a canal Panama" ✓
- "hello" ✗

### Features

- Case-insensitive checking
- Ignores spaces and punctuation
- Interactive testing mode
- Built-in example cases
